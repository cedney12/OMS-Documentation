Order Management

This project is meant to replace the current solution for purchase order management.

Current Solution

The current method of recording order data is in the form of an excel spreadsheet, with manual entry of all fields and no rigid formatting or misinput handling. This is where the current spreadsheet resides here: [Order Management Spreadsheet]{https://intercotradingco.sharepoint.com/:x:/g/EdAdIL_Ks1NDh4TioSRgBFgBRD_Nzoy9f-n4qX4LOWX2YA?e=eQe1yW}

Proposed Solution, and This Project

This project is built in React and proposes a way to display the data in a manner similar to the spreadsheet, but adding better functionality to keep data uniform.

Functionality

The React web app has the following functionality:

  • Data Grid with pagination, searching, sorting, and filtering by facility
  • Viewing Supplier and Contact data in a modal
  • Creating and Editing Orders in a modal
  • Report Generation for Bills of Ladings and Rate & Load Confirmations
