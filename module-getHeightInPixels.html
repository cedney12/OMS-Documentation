getHeightInPixels

Returns an integer value based on the number of pixels. Currently supports the following units:

  • px
  • vh
  • vw
  • %
  • <> (nothing, i.e. just a number)
Parameters:
Name Type Attributes Description
rawHeight string

Height as a CSS rule
elementRef React.MutableRefObject.<any> <optional>

Reference to the element trying to size, used in % cases
