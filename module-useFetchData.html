useFetchData

Custom hook that fetches data from the given URL, with options to sort the data by a certain field (sortField) and to prioritize a value in that field or another field (priorityValue & priorityField)

Parameters:
Name Type Attributes Description
fetchUrl string

URL for the API call
options object

Options for the API call, default {}
sortField string <optional>

Field by which to sort the resulting data
sortDirection "asc" | "desc" <optional>

Direction by which to sort
priorityField string <optional>

Field by which to prioritize certain items
priorityValue string <optional>

Value to prioritize in the
